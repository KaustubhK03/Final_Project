Extraction of Traffic Data

A Modern Computer Vision Model whose main task is to gather infromation from Traffic videos. It counts the number of objects and also the type of objects that crosses a user defined line in the video and Gives the output as a CSV.

Key Features

Detecting vehicles and Pedestrians

This Model is suitable to detect the number of Pedestrians and Vehicles in each and every frame

Counts the Objects.

Increases the aggregate count when an object crosses the line.

Identifies it's direction

Additional to increasing the aggregate count it also detects the direction from which the object crossed the line.

Outputs a csv

Combining all of these features, the model outputs all the information in a csv format. the csv contains ["Frame_No", "ObjID", "x1", "y1", "x2", "y2", "Class_Name", "Confidence", "Direction", "Aggregate_Count"]
